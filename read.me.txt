ЗАПУСК
Для запуска программы необходимо создать новый проект java. В новый проект скопировать файл с кодом прогаммы и открыть его.

ВХОДНЫЕ ДАННЫЕ
Для тестирования программы необходимо ввести следующие данные: 
1) количество вершин графа
2) количество дуг графа (если дуга двусторонняя, ее необходимо учесть дважды)
3) дуги графа в следующем виде:
	a b q
	где а - исходящая вершина
	b - входящая вершина
	q - вес дуги
4) вершина, из которой начинается движение, и вершина, в которой заканчивается движение

ОЦЕНКА СЛОЖНОСТИ АЛГОРИТМА
В оценке учитывается ТОЛЬКО сам алгоритм Дейкстры.
В первую очередь в агоритме присутствует цикл while. Выполнение идет n раз.
В цикле while присутсвует цикл for и функция arraymin.
Цикл for выполняется n раз.
Рассмотрим функцию arraymin: в ней присутсвует два цикла for, которые выполняются по n раз.
Получаем: n*(n + 2n) = О(n^2)
